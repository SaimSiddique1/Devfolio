# Security Policy

## Supported Versions

The template is maintained on a best-effort basis.  
Security patches will be applied to the latest version.

## Reporting a Vulnerability

If you discover a security issue, please email:

**Trevinoamelia25@gmail.com**

Please include:

- A description of the vulnerability  
- Steps to reproduce  
- Potential impact  
- Suggested fixes (optional)

We will review your report promptly and privately.
